# Kheyre Electronics Website

## Magaca Mashruuca
Kheyre Electronics Website

## Sharaxaad
Mashruucani waa website loogu talagalay dukaanka Kheyre Electronics.
Website-ku wuxuu soo bandhigayaa xogta shirkadda, adeegyada ay bixiso iyo habka loola xiriiro.

## Bogagga Website-ka

- Home (index.html)
- About Us (about.html)
- Services (services.html)
- Contact Us (contact.html)

## Teknoolojiyadda La Adeegsaday

- HTML5
- CSS3

## Astaamaha Website-ka

- Responsive Design
- Mobile Friendly
- Tablet Friendly
- Laptop/Desktop Friendly
- Navigation Menu
- Hero Section
- About Section
- Services Section
- Contact Form
- Footer

## Sida Loo Isticmaalo

1. Soo dejiso dhammaan faylasha.
2. Fur galka mashruuca adigoo isticmaalaya Visual Studio Code.
3. Fur `index.html`.
4. Riix **Go Live** ama ku fur Browser-ka.
